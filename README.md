# 🌐 Tripzomania – Luxury Travel Club Website

Live Website: https://tripzomania.com/

## 📌 Overview

Tripzomania is a modern, responsive luxury travel club website built to connect travelers with premium vacation packages, resort getaways, and VIP travel memberships. The website features an immersive 3D experience with a clean, professional UI focused on converting visitors into members through compelling visuals and strong call-to-action sections.

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Immersive 3D background using Three.js
- Modern luxury UI/UX with gold & navy branding
- Conversion-focused landing page structure
- Sections include:
  - Hero with video background
  - About / Vision
  - VIP Membership Benefits
  - Vista Cay Luxury Resort
  - The Club At Maingate
  - Elite Travel Partners
  - Member Testimonials
  - Contact & FAQ
- Interactive 3D tilt cards
- Live concierge chat widget
- FAQ accordion
- Modal popups for resort details
- SEO-friendly structure
- Smooth scroll animations with GSAP & ScrollTrigger

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Three.js (3D background & animations)
- GSAP + ScrollTrigger (scroll animations)
- Tailwind CSS (utility-first styling)
- Font Awesome (icons)
- Google Fonts (Playfair Display, Inter)

## 🎯 Purpose

The purpose of this project is to demonstrate advanced frontend development skills by building a real-world luxury travel website. It focuses on 3D web experiences, immersive UI design, responsiveness, and lead generation strategy.

## 📈 Key Highlights

- Immersive 3D Three.js animated background
- Luxury gold & navy branding throughout
- Interactive card tilt effects on hover
- Business-oriented design with strong UX
- Mobile-first responsive layout
- Professional travel club branding
